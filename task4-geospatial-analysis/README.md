# Geospatial Data Analysis Project

## Project Overview

This project focuses on analyzing location-based sales data to identify high-performing countries, regions, and markets.

Geospatial data analysis helps businesses understand:
- regional sales performance
- market distribution
- expansion opportunities
- location-based trends

The analysis was performed using Python, Pandas, Matplotlib, and Seaborn.

---

# Objective

The main objectives of this project are:

- Analyze sales performance by country
- Evaluate regional sales distribution
- Identify high-performing markets
- Generate business insights
- Visualize geographical sales trends
